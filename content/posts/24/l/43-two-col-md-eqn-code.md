+ Roots of $y = ax^2 + bx + c$ can be obtained from
$$
x_{1,2} = \frac{-b \pm \sqrt{D}}{2a}.
$$
+ Discriminant $D = b^2 - 4c$. 
{{< /twocolumn >}}

{{< twocolumn col="right" markdown="yes" >}}
```py
a = 1
b = -9.5
c = 15

D = b**2 - 4*a*c
x1 = (-b - D**0.5) / (2*a)
x2 = (-b + D**0.5) / (2*a)

print('x1 =', x1)
print('x2 =', x2)
```
